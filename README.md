## 🚀 Getting Started

### Installation
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Create Your First Project
```bash
cargo new hello_rust
cd hello_rust
cargo run
```

### Update Rust
```bash
rustup update
```
